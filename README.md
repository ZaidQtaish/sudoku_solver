# Sudoku Solver

A full-stack Sudoku solver application with a React frontend and Express backend.

## Tech Stack

- **Frontend:** React, Vite
- **Backend:** Node.js, Express
- **Testing:** Mocha, Chai

## Getting Started

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
